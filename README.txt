Dieser Ordner beinhaltet die komplette FH-Nav Dortmund App mit allen Abhängigkeiten.

Da das Projekt eigentlich nie für eine Veröffentlichung vorgesehen wurde, sind Kommentare und auch Dokumentation nicht entsprechend.
Wer trotzdem Lust hat die App weiterzuentwickeln ist herzlich eingeladen.


Das Projekt sollte recht einfach mit Eclipse bearbeitet werden können.

Vorraussetzungen:

 - Java
 - AndroidSDK
 - Eclipse Plugin für Android (falls Eclipse benutzt wird)
